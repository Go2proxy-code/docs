* API介面位址：https://api.go2proxy.com/v1/static/proxies

* 請求方式：GET

* QPS限制：單訂單1秒訪問1次

## 1.參數

| 參數名       | 必填 | 類型       | 說明             |
| :-------- | :- | :------- | :------------- |
| app_id   | 是  | `string` | API存取憑證        |
| signature | 是  | `string` | API存取憑證        |
| orderid   | 是  | `string` | 訂單號            |
| ft       | 否  | `string`    | 返回格式，預設為json <br />可選項（json,text,xml）    |
| delimiter    | 否  | `string`    | 分隔符，當ft=text時必填<br />可選項（\r\n,\n,space,|） <br />或自訂義"custom_t" t為自訂義值      |
| display    | 否  | `string` | 返回結果，預設返回ip:port<br /> display=ip_port_user_pass時返回ip、port、使用者名稱、密碼<br />display=ip_port_user_pass_loc時返回ip、port、使用者名稱、密碼、地區|


## 2.回傳參數說明

| 參數名  | 類型          | 說明                           |
| :--- | :---------- | :--------------------------- |
| code | `int`       | 請求業務狀態，0為成功，非0均為業務失敗，錯誤碼詳情參下 |
| msg  | `string`    | 請求業務描述              |
| data | `"" or map` | 請求業務失敗時為`空字串`                |
