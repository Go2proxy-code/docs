## 1.Dynamic Residential Proxy
| HTTP Status Code  | Reason Phrase| illustrate   |
|:-----------  |:-----------------|:---------|
|`407` |Proxy Authentication Failed/White IP Failed|Username, password or whitelist error|
|`413` |Illegal Request|Request format error|
|`415` |Request Over Limit|Concurrency exceeded|
|`416` |Bandwidth Over Limit|Bandwidth exceeded|
|`418` |Miss Host|Unable to resolve destination host address|
|`419` |Mainland China Access Not Permitted|Mainland China client banned|
|`421` |DNS Resolution Failed|DNS resolution failed|
|`423` |Public Host Required|Target host address non-public network ip|
|`429` |Insufficient Traffic|Insufficient order flow|
|`431` |Unauthorized IP Usage|Use unauthorized ip|
|`503` |Proxy Connect Error|Failed to connect to target host|
