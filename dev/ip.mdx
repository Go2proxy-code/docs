## 1.管理台设置
您可以在控制台 [「我的訂單」](https://www.go2proxy.com/console/orders) 裡查看和修改當前的地區和周期配置。修改成功後，變更將在一分鐘內生效。

## 2.编码设置
- 以使用美國紐約地區的IP，並將IP更換周期設置為1分鐘為例。
- 您可以在程式中通過以下方式進行設置：
`{username}-region-{US}-area-{New York}-period-{1}:{password}`， 將用戶名、國家編碼、地區編碼、IP更換周期(分鐘)、密碼填入。例如用戶名`gf2814951572`，密碼`aft415oyt`，這次請求使用`美國-紐約地區ip、IP更換周期1分鐘`。

```python 程式碼範例
gf2814951572-region-US-area-New York-period-1:aft415oyt
```

<Note>當您通過編碼方式設置地區和周期時，系統會忽略您在控制台 [「我的訂單」](https://www.go2proxy.com/console/orders) 裡的地區和周期配置。</Note>
